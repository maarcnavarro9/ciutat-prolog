# ciutat-prolog

## How to run ⚙️

1. Go to this website

    `https://swish.swi-prolog.org`

2. Copy & paste the code in "ciutat.pl"
   
3. Run prompts

    `ciutats([2,1], [1,2], [2,1], [1,2]).`
    `ciutats([2,2,1,3], [3,1,3,2], [3,1,2,2], [2,3,1,2]).`
    `ciutats([2,2,2,2,1],[4,3,2,1,5],[5,1,2,3,4],[1,2,2,2,2]).`
